# agentic-rpa-document-orchestrator
My project introduces an agentic AI layer above traditional RPA systems. Instead of fixed workflows, the AI agent interprets user goals, plans execution strategies, dynamically selects RPA bots, and adapts to failures. This makes automation more flexible and intelligent.
# Abstract
Traditional Robotic Process Automation (RPA) systems operate on static, rule-based workflows and require manual intervention whenever task requirements or execution environments change. This limits their adaptability in dynamic, real-world scenarios, especially when handling unstructured inputs such as documents received via email. To address this limitation, this project proposes an Agentic AI–Driven Intelligent RPA Task Orchestrator, which integrates a decision-making agent with RPA workflows to enable dynamic and context-aware task execution.

The proposed system introduces an agentic layer capable of interpreting document content and determining appropriate processing strategies. Instead of predefined linear workflows, the agent analyzes OCR-extracted text, identifies document characteristics, and dynamically selects suitable automation steps for processing. The system monitors incoming emails, extracts document attachments, converts them into machine-readable text using OCR, and uses a rule-based decision module to guide workflow execution. Execution logs and decision traces are maintained to ensure transparency and traceability.

The system is implemented using Python-based decision logic for agent behavior and UiPath-based RPA workflows for execution. Offline OCR engines such as Microsoft OCR or Tesseract are used to extract textual data from documents. The architecture follows a modular design, separating ingestion, preprocessing, decision-making, and execution layers to ensure scalability and maintainability.

The outcome of this project is a functional prototype demonstrating how an agentic decision layer can enhance traditional RPA systems by enabling adaptive and semi-autonomous document processing. The system reduces manual intervention, improves processing efficiency, and provides a foundation for future integration of more advanced intelligent components.
